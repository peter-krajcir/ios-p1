Project 1 - Pitch Perfect APP
=============================

Run the project
---------------
In order to run the project, you need to fork/clone/download the project and open the file `PitchPerfect.xcodeproj`. You can also visit my github webapge and get the latest version. (https://github.com/peter-krajcir/ios-p1)

Description
-----------
The Pitch Perfect app allows users to record a sound using the deviceâ€™s microphone. It then allows users to play the recorded sound back with four different sound modulations: Chipmunk, Darth Vader, Super Slow, and Super Fast. It has 2 additional sound filter - reverb and echo. Reverb uses Cathedral Preset with 75% mix and echo is delayed by 0.2 second.
